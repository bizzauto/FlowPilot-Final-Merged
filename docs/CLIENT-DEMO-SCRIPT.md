Final Client Demo Script

Demo length:
10 to 15 minutes

Pre-demo checklist:

App running
Admin login ready
Demo data generated
n8n open
WhatsApp test number ready
Razorpay test mode ready
Client slug ready
Mobile responsive checked

Demo flow:

1. Open landing page.
Say: This is the marketing website.

2. Open pricing page.
Say: Plans and billing are managed here.

3. Login.
Say: This is secure SaaS login.

4. Open /welcome.
Say: This is onboarding checklist.

5. Open /dashboard.
Say: This shows real analytics.
Click Generate Demo Data.

6. Open /contacts.
Say: This is CRM.
Show add contact, import CSV, export CSV.

7. Open /pipeline.
Say: This is sales pipeline.

8. Open /inbox.
Say: This is unified inbox for WhatsApp conversations.

9. Open /broadcast.
Say: This is WhatsApp broadcast composer.
Create campaign and queue.

10. Open /automation-builder.
Say: This is drag-drop automation builder.
Add trigger, wait, send WhatsApp, add tag.

11. Open /analytics.
Say: This shows advanced analytics.

12. Open /support.
Say: This is support ticket system.
Create ticket.

13. Open /agency-wizard.
Say: This creates client workspace.
Create client.

14. Open /l/client-slug.
Say: This is white-label landing page.

15. Open /billing-portal.
Say: Client can manage subscription.

16. Open /settings.
Show WhatsApp, API keys, team, agency, white-label, billing, deployment.

17. Open n8n.
Say: n8n is connected for advanced automation.

18. Resize browser to mobile.
Say: App is mobile responsive and PWA ready.

This is a complete SaaS automation platform.
