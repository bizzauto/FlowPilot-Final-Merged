Admin Onboarding Flow

1. Login as admin.

2. Open:
   /welcome

3. Complete checklist:
   - Database connected
   - Redis connected
   - NextAuth secret set
   - WhatsApp configured
   - Stripe configured
   - n8n webhook configured
   - Create first contact
   - Create API key

4. Create API key:
   /settings/api-keys

5. Add API key to n8n.

6. Configure WhatsApp:
   /settings/whatsapp

7. Create templates:
   /settings/whatsapp-templates

8. Create campaign:
   /campaigns

9. Create automation:
   /automations

10. Check production checklist:
   /settings/production
